
```

```

**Действия агента:**
```bash

  "SELECT id, telegram_payment_id, status FROM transactions ORDER BY id DESC LIMIT 5;"
```

---

### **4. Идемпотентность**

**Лог:**
```
⚠️ Платёж уже обработан: {charge_id} (транзакция {id})
```

**Если дубликат:**
```
❌ Duplicate transaction created!
```

**Действия агента:**
```bash
# Проверка дубликатов
sqlite3 /var/www/tarot-miniapp/server/tarot.db \
  "SELECT telegram_payment_id, COUNT(*) as count FROM transactions GROUP BY telegram_payment_id HAVING count > 1;"
```
