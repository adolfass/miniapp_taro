## 1. НАЗНАЧЕНИЕ ПРОЕКТА

**Telegram Mini App для гадания на картах Таро**

### 1.1 Основная цель
Мистическое веб-приложение внутри Telegram для гадания на картах Таро с:
- Визуальным восприятием карт
- Плавными анимациями (120 Гц)
- Тактильным откликом
- Звуковым сопровождением
- Оплатой через Telegram Stars
- Админ-панелью для управления

### 1.2 Платформа
- **Telegram WebApp** — работает внутри Telegram
- **Адаптивный дизайн** — ПК / Мобильные устройства
- **Vanilla JavaScript** — без фреймворков (полный контроль)

### 2.2 Расклады

#### **Ежедневный расклад (3 карты)**

**Механика:**
1. Тасовка колоды (6 секунд)
2. Выбор 3 случайных карт
3. Переворот по тапу с частицами
4. Позиции:
   - Прошлое
   - Настоящее
   - Будущее
5. Кнопки действий:
   - «Поделиться с тарологом»
   - «Новый расклад»

**Данные карт:**
- 78 карт (22 Старших Аркана + 56 Младших)
- Изображения WebP
- Описание + ключевые слова

---

#### **Расклад «Путь» (7 карт, Хайо Банцхаф)**

**Механика:**
1. Тасовка колоды
2. **7-секундная концентрация** (песочные часы)
3. **Активация:**
   - Кнопка «Сделать расклад»
   - **ИЛИ** встряхивание телефона (акселерометр)
4. Схема расклада:
   ```
       1 (Суть вопроса)
     2   7
     3   6
     4   5
   ```
5. Позиции:
   - 1: Суть вопроса (центр)
   - 2-4: Прошлое (ментальный, астральный, физический)
   - 5-7: Будущее (физический, астральный, ментальный)

**Особенности:**
- Шахматное расположение карт
- Общее толкование с учётом позиций
- Анимация песочных часов (7 секунд)

### 2.3 Telegram Stars интеграция

**Поток пользователя:**

```
1. Расклад готов
       ↓
2. Кнопка «Поделиться с тарологом»
       ↓
3. Экран выбора таролога (3 карточки)
       ↓
4. Выбор таролога → Модальное окно оплаты
       ↓
5. Оплата через Telegram Stars API
       ↓
6. Успешная оплата → Чат с тарологом (25 мин)
       ↓
7. Таймер чата → Окончание консультации
       ↓
8. Экран оценки таролога (1-5 звёзд)
```

**Распределение платежа:**
- **10%** — разработчик (комиссия платформы)
- **90%** — таролог

**Динамическое ценообразование:**
```javascript
function calculatePrice(sessionsCompleted) {
  const level = Math.floor(sessionsCompleted / 10) + 1;
  const price = 33 * Math.pow(1.1, level - 1);
  return Math.min(Math.round(price), 333); // Максимум 333 ⭐
}
```

### 3.4 Анимации

#### **Переворот карты**

**Особенности:**

#### **Частицы при перевороте (Canvas)**

**Параметры:**
- 20 частиц на переворот
- Цвет: золотой (`hsl(45-60, 80%, 60-80%)`)
- Физика: гравитация (`vy += 0.1`)
- Затухание: `decay 0.02-0.04`
- Время жизни: ~2 секунды

**Код:**
```javascript
import { createParticles } from './animations.js';

setTimeout(() => {
  const rect = cardEl.getBoundingClientRect();
  createParticles(
    rect.left + rect.width / 2,
    rect.top + rect.height / 2
  );
}, 350); // Середина анимации
```

**Параметры:**
- Canvas на весь экран
- Количество звёзд: `area / 4000`

- Скорость: 0.01-0.03

**Код:**

### 6.3 Микроцикл диагностики (Сравнение версий)

**Критерии РАБОТАЮЩЕЙ версии кода:**

```
✅ Код выгружен Qwen Code на GitHub
✅ Задеплоен Opencode Agent на сервер
✅ Протестирован Opencode Agent (все тесты ✅)
✅ Протестирован Менеджером (в приложении + web + desktop)
✅ Зафиксирован в toQwen.md как "ГОТОВО"
✅ Помечен Git Tag "RELEASE"
```

---

**Алгоритм при обнаружении бага:**

```bash
# 1. Найти последнюю рабочую версию
git log --oneline | grep "RELEASE"
# или
git tag -l "RELEASE*"

# 2. Сравнить с текущей версией
git diff RELEASE-1.0..HEAD -- path/to/file.js

# 3. Если найдены изменения → Откатить → Тестировать
git checkout RELEASE-1.0 -- path/to/file.js

# 4. Если помогло → Исправить → Закоммитить
# 5. Если нет → Глубокая диагностика (логи, консоль)
```

---

**Пример использования:**

```
СИТУАЦИЯ: После обновления кнопки админки не работают

ШАГИ:
1. Opencode Agent сообщает о баге в toQwen.md
2. Qwen Code находит последний RELEASE:
   git tag -l "RELEASE*" → RELEASE-1.2
3. Сравнивает изменения:
   git diff RELEASE-1.2..HEAD -- public/js/admin.js
4. Находит изменение в setupEventListeners()
5. Откатывает:
   git checkout RELEASE-1.2 -- public/js/admin.js
6. Тестирует → Работает!
7. Исправляет код → Коммит → Push
8. Новый цикл тестирования
```

---

**Преимущества микроцикла:**

| Преимущество | Описание |

| Восстановление | Можно откатиться к рабочей версии |

**Ограничения:**
