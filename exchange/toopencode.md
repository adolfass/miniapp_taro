## 📋 ДЕТАЛЬНЫЙ ПЛАН ТЕСТИРОВАНИЯ

### Этап 1: Подготовка (5 минут)

**1.1. Проверка сервера:**
```bash
pm2 status
pm2 logs tarot-app --lines 20
```

**1.2. Проверка БД:**
```bash
sqlite3 database.db "SELECT COUNT(*) FROM tarologists;"
sqlite3 database.db "SELECT COUNT(*) FROM users;"
```

**1.3. Тестовые данные:**
- Таролог: ID 1 (Роман)
- Пользователь: Telegram ID менеджера
- Цена: 48 ⭐ (или TEST_PRICE=1)

---

### Этап 2: Оплата консультации (10 минут)

**2.1. Открыть TMA в Telegram:**
- Запустить бота @goldentarot_bot
- Нажать "Открыть приложение"

**2.2. Выбрать таролога:**
- Найти карточку таролога "Роман"
- Нажать "Выбрать"

**2.3. Оплатить консультацию:**
- Нажать "Оплатить (48 ⭐)"
- Подтвердить оплату в Telegram
- **Зафиксировать:** статус оплаты, номер транзакции

**2.4. Проверка в БД:**
```bash
sqlite3 database.db "SELECT id, status, stars_amount, session_id FROM transactions ORDER BY created_at DESC LIMIT 1;"
```

**Ожидаемый результат:**
- ✅ Статус: `paid`
- ✅ session_id: НЕ NULL
- ✅ stars_amount: 48 (или 1 в тесте)

---

### Этап 3: Чат между клиентом и тарологом (30 минут)

**3.1. Подключение клиента:**
- Открыть чат в приложении
- Проверить что видно таймер (25:00)
- Отправить первое сообщение

**3.2. Подключение таролога:**
- Войти в админку (/admin.html)
- Открыть вкладку "Чаты"
- Найти активную сессию
- Подключиться к чату

**3.3. Диалог (минимум 10 сообщений):**
```
Клиент: "Здравствуйте! Хочу получить консультацию..."
Таролог: "Добрый день! Расскажите что вас интересует..."
Клиент: "Меня волнует вопрос отношений..."
Таролог: "Понимаю. Давайте разберём ситуацию..."
... (ещё 6 сообщений)
```

**3.4. Проверка в БД:**
```bash
sqlite3 database.db "SELECT COUNT(*) FROM messages WHERE session_id = [ID_СЕССИИ];"
```

**Ожидаемый результат:**
- ✅ Сообщений: >= 10
- ✅ Таймер уменьшается
- ✅ Сообщения видны обеим сторонам

---

### Этап 4: Завершение консультации (5 минут)

**4.1. Ожидание 25 минут ИЛИ ручное завершение:**
- Дождаться истечения таймера
- ИЛИ таролог нажимает "Завершить консультацию"

**4.2. Проверка в БД:**
```bash
sqlite3 database.db "SELECT active, completed FROM chat_sessions WHERE id = [ID_СЕССИИ];"
```

**Ожидаемый результат:**
- ✅ active: 0
- ✅ completed: 1

---

### Этап 5: Система оценок (5 минут)

**5.1. Экран оценки:**
- После завершения появляется экран "Оцените таролога"
- 5 звёзд (1-5)

**5.2. Отправить оценку:**
- Нажать 5 звёзд
- Написать отзыв (опционально)
- Отправить

**5.3. Проверка в БД:**
```bash
sqlite3 database.db "SELECT rating, total_ratings FROM tarologists WHERE id = 1;"
```

**Ожидаемый результат:**
- ✅ rating: обновился
- ✅ total_ratings: +1

---

### Этап 6: Автовозврат (15 минут)

**6.1. Создать транзакцию БЕЗ чата:**
- Оплатить консультацию
- НЕ начинать чат (не писать сообщения)

**6.2. Ожидание 10 минут:**
- Запустить таймер
- Ждать срабатывания cron (каждые 5 минут)

**6.3. Проверка возврата:**
```bash
sqlite3 database.db "SELECT status, auto_refund_processed, auto_refund_reason FROM transactions ORDER BY created_at DESC LIMIT 1;"
```

**Ожидаемый результат:**
- ✅ status: `refunded`
- ✅ auto_refund_processed: 1
- ✅ auto_refund_reason: `NO_MESSAGES`

**6.4. Проверка уведомления:**
- Проверить бота @goldentarot_bot
- Должно прийти сообщение о возврате

---

### Этап 7: Интерфейс админки (15 минут)

- [ ] Ошибки отображаются
- [ ] Мобильная версия работает
